# RockPaperScissors_TCP_Single_Multi_MODE
가위 바위 보 TCP 프로젝트

## Get Started
- 1 Run TCP Server:
```sh
   java MultiServer.java 
   java SingleServer.java
   ``` 
- 2 Run GUI:
```sh
   java StartFrame.java
   ```


### Built with:
- Design Template: Figma
- GUI: Java Swing
- Language: Java
- Connection: TCP
